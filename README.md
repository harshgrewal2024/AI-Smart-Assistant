# 🤖 AI Smart Assistant

An intelligent full-stack web application that combines an AI chatbot and a resume analyzer to help users interact and improve their resumes.

---

## 🚀 Features

### 🔐 Authentication
- User Signup & Login system
- Secure password hashing using Werkzeug

### 💬 AI Chatbot
- Real-time AI chat using OpenRouter API
- Chat history stored in database
- Typing animation for realistic experience

### 📄 Resume Analyzer
- Upload PDF resumes
- Extract text using PyPDF2
- Score based on skills & keywords
- Visual feedback with progress bar

### 📊 Dashboard
- View chat history
- View resume analysis history
- Clean and modern UI

---

## 🛠 Tech Stack

- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Backend:** Flask (Python)  
- **Database:** SQLite  
- **AI API:** OpenRouter (LLM integration)  
- **Other Tools:** PyPDF2, Werkzeug
